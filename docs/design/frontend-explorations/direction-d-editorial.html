值得带

单台炮击战斗服（未移动、重型磁轨枪 2 发）对帝国骑士每轮期望约 2.3 伤，其中约 1.0 来自[毁灭伤害]致命伤；满编 3 台一轮约 7 伤，约 3 轮打掉一台满血骑士（22W）。值得带：80 分/台的 S12 AP-4 D6+1，是钛帝国点数效率最高的反坦克平台之一。

The Math
  1. [重型]2：未移动 +1 命中 → 3+ 命中（67%）
  2. S12 对 T12 → 4+ 穿防（50%）
  3. AP-4 击穿 3+ 甲，骑士只能用 5++ 无效保护 → 失防 67%
  4. 穿防骰 6 触发[毁灭伤害]3 → 直接致命伤，无视任何保护
  5. D6+1 平均 4.5 伤/发；另有寻觅者导弹（一次性，S14 AP-3）可补一发
