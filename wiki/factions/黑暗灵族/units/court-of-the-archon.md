## 技能
- **Deadly Retinue (Pain)**：At the start of your Shooting phase or at the start of the Fight phase, you can spend 1 Pain token to Empower this unit. While this unit is Empowered:<br><ul><li>If it contains one or more Lhamaean models, melee weapons equipped by models in this unit have the [LETHAL HITS] ability.</li><li>If it contains one or more Medusae models, ranged weapons equipped by models in this unit have the [IGNORES COVER] ability.</li><li>If it contains one or more Sslyth models, each time a melee attack targets this unit, subtract 1 from the Wound roll.</li><li>If this unit contains one or more Ur-ghul models, this unit has the Fights First ability.</li></ul>
- **Court of the Archon**：While a <span class="kwb">CHARACTER</span> model is leading a unit that contains one or more Lhamaean, Medusae, Sslyth and/or Ur-ghul models, that <span class="kwb">CHARACTER</span> model has the Feel No Pain 4+ ability.
- **COURT OF THE ARCHON**：This unit can be led by an Archon. Alternatively, in the Declare Battle Formations step, this unit can join one [[factions/黑暗灵族/units/kabalite-warriors.md|Kabalite Warriors]] or [[factions/黑暗灵族/units/hand-of-the-archon.md|Hand of the Archon]] unit from your army that is being led by an <span class="kwb">ARCHON</span> (a unit cannot have more than one <span class="kwb">COURT</span> <span class="kwb2">OF</span> <span class="kwb2">THE</span> <span class="kwb">ARCHON</span> unit joined to it). If it does, until the end of the battle, every model in this unit counts as part of that <span class="kwb">KABALITE</span> <span class="kwb">WARRIORS</span> or <span class="kwb">HAND</span> <span class="kwb2">OF</span> <span class="kwb2">THE</span> <span class="kwb">ARCHON</span> unit, and its Starting Strength is increased accordingly.
