造伤骰是攻击序列第二步，用武器 S 值对比目标 T 值判定是否造伤。

## 造伤骰 WOUND ROLL

为每次成功命中投 1 个 D6 作为造伤骰，按**从上到下第一条适用**的条件判定（05.02）：

1. 未修正结果为 1：**失败**（永远失败）。
2. 未修正结果为 6：**造伤暴击**（critical wound，永远造伤，并可触发【毁灭伤害】等规则）。
3. 结果不小于下表要求：造伤。
4. 其他结果：失败。

| 武器 S 对比目标 T | 需求 |
|---|---|
| S 为 T 的两倍或以上 | 2+ |
| S 高于 T | 3+ |
| S 等于 T | 4+ |
| S 低于 T | 5+ |
| S 为 T 的一半或以下 | 6+ |

> 11 版语义与十版一致（对比表、未修正 1/6 规则均不变）。十版"造伤骰修正上限 ±1"的条款未见于 11 版核心规则书正文，本页不作断言。[11版 Core Rules 05.02]
