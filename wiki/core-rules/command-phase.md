指挥阶段双方获取 CP，随后主动玩家检查部队士气。

## 指挥阶段 COMMAND PHASE

按以下步骤结算（08.01-08.05）：

1. **阶段开始**：结算此时触发的规则。
2. **获取核心 CP**：双方玩家各获得 1 点指挥点数（CP），用于使用计谋（15 章）。
3. **战斗震慑**：主动玩家为己方每个满足以下任一条件的单位掷一次[[core-rules/battle-shock-test.md|战斗震慑骰]]（01.07）：① 该单位当前已被战斗震慑；② 该单位处于**半数及以下战力**（at or below half-strength）。若单位在本步开始时处于震慑状态且本次投骰成功，则**解除震慑**。
4. **指挥技能**：结算在指挥阶段触发的其他规则。
5. **阶段结束**：先结算非任务规则，再结算任务规则。

> 11 版变更（2026-06-20 起）：战斗震慑步重做——已震慑单位每回合重掷、成功即解除；触发线为"半数及以下战力"；并新增独立的"指挥技能"结算步。[11版 Core Rules 08.01-08.05]
